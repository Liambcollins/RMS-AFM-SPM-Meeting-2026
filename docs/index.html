Repository Overview

This repository accompanies the Machine Learning in SPM tutorial at the RMS AFM & SPM Meeting 2026. It contains notebooks, synthetic datasets, and example workflows demonstrating how ML can enhance modern SPM experiments.

📦

Portable Demos

Synthetic SPM‑like datasets allow all examples to run anywhere — no instrument access or proprietary file formats required.

📓

Notebook‑First

Interactive Jupyter notebooks provide visual explanations and runnable demonstrations, suitable for live demo or self‑study.

🧠

Practical ML

Examples focus on real SPM problems — noisy hyperspectral data, domain mapping, and signal recovery — using only common Python packages.

🔮

Future SPM

Later modules explore smart measurements, adaptive sampling, and autonomous microscopy — the frontier of intelligent instrumentation.

Tutorial Modules

The tutorial progresses from simple ML tools to advanced autonomous measurement concepts. Each notebook is self‑contained and runs on synthetic data.

00

Setup & Overview

Motivation for ML in SPM, data types encountered in the field, and a tour of the tutorial structure. Includes environment checks.

00_setup_and_overview.ipynb

01

PCA Denoising

Use dimensionality reduction to separate signal from noise in synthetic hyperspectral SPM data. Includes scree plots, component maps, and reconstruction comparisons.

01_pca_denoising.ipynb

02

Clustering & Phase Mapping

Automatically discover domains and material regions using KMeans and Gaussian Mixture Models. Includes soft‑assignment heatmaps and cluster quality metrics.

02_clustering_phase_mapping.ipynb

03

Sparse & Smart Measurements

Demonstrate how fewer measurements can still recover spatial structure. Compare random, grid, and gradient‑guided sampling strategies with reconstruction quality curves.

03_sparse_smart_measurements.ipynb

04

Autonomous SPM · Flagship Demo

Implement a toy closed‑loop adaptive measurement loop. The virtual instrument decides where to measure next based on uncertainty and gradient scores, then updates its model estimate. Visualise the measurement path and compare adaptive vs random sampling — this is the future of intelligent microscopy.

04_autonomous_spm.ipynb

Getting Started

All notebooks run on synthetic data. No instrument access or proprietary files needed.

1 · Clone

git clone https://github.com/Liambcollins/RMS-AFM-SPM-Meeting-2026.git
cd rms-spm-ml-tutorial-2026

2 · Create environment

# With conda (recommended)
conda env create -f environment.yml
conda activate spm-ml-tutorial

# Or with pip
pip install -r requirements.txt

3 · Launch JupyterLab

jupyter lab

Then open any notebook in the notebooks/ folder.
